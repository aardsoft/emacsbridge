\doxysubsection{Detailed Description}
This is the main class for the service process. Per default it sets up a connection to the service using Qt remote objects, and exposes select service A\+P\+Is to Q\+ML. It is also possible to use the dummy constructor to test loading of Q\+ML components without providing the full backend logic.
