\doxysubsection{Detailed Description}
This class handles logging of messages from the Qt log functions to various useful outputs. Depending on the system this may be one or more of the following\+:
\begin{DoxyItemize}
\item stdout/stderr
\item systemd journal
\item Android log, readable via logcat
\item internal ringbuffer, readable via emacsbridge -\/log
\end{DoxyItemize}

This class is a singleton, and most methods are static for ease of use. Additionally it is possible to inject raw log entries for parsing, which generally is usable to forward messages from multiple processes to a single logging instance.
